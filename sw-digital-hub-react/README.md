# SW Digital Hub Corporate Website

Premium responsive static corporate website built with React + Vite.

## Features

- Premium modern UI
- Fully responsive layout
- SEO-friendly meta tags
- Services, portfolio, about, process, stats, and enquiry sections
- Framer Motion animations
- Easy to deploy on Vercel, Netlify, or GitHub Pages

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy on Vercel

1. Push this project to GitHub.
2. Open Vercel and import the GitHub repository.
3. Framework: Vite.
4. Build command: `npm run build`.
5. Output directory: `dist`.
6. Click Deploy.

## Edit Content

Main website text, services, portfolio, stats, and process data are inside:

```bash
src/data/content.js
```
